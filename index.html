Createimages.js

A jQuery plugin that creates image placeholders to semplify front end developing. You can choose an online service fallback.
It's objective is to be as less invasive as possible. With it you can create the placeholders without adding any class or altering your html in any way. All you have to do is include the script and set the source of the images to your favourit Placeholding webservice (it defaults to http://lorempixel.com).

example:
img src="http://lorempixel/400/400/animals" alt="placeholder"
Renders a 400x400 image with an animal on it.
If no category is specified it will render a gray image.
